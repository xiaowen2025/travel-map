## 使用说明

本系列文档以时间为主线，以景点为锚点，旨在帮助旅行者在实地探访欧洲各地的历史遗址时，能够将单个景点串联成一部连贯的欧洲文明史。

**每章节的结构：**
1. **时代概述**：简述该时期的历史背景和核心特征
2. **景点详解**：每个景点包含位置、年代、核心价值和历史意义
3. **景点地图**：汇总该时期所有景点的位置、年代和UNESCO状态
4. **时代概要**：总结该时期对欧洲历史的深远影响

**建议行程搭配：**
- 史前与古典时代 → 希腊 + 意大利
- 中世纪与贸易 → 法国 + 德国 + 克罗地亚
- 文艺复兴与宗教 → 意大利 + 德国 + 荷兰
- 帝国与革命 → 法国 + 英国 + 奥地利
- 世界大战与现代 → 法国 + 德国 + 波兰
